# Individual Ability
# Cocoon

- [[Advisor]] can put up a defensive barrier if they want to remain an Advisor for longer

- gain tomato resistance
- gain Health Regeneration

- costs a % of max points / Minute

- Each Advisor can decide if and when to use this ability



# Council Abilities

- Advisors vote on activating or deactivating these abilities

# Offensive
## No Mercy

- increases Combine Attack Rate

- costs Points / Minute for all Advisors

## Logistics

- Reduce Combine Attack Travel Time

# Defensive

## Cauterize

- increases Combine HP Regeneration

- costs Points / Minute for all Advisors
# Stasis

- freezes Earth stock market for x seconds

- costs a one time amount of points split between advisors

# Propaganda

^f8e434

- [[Chat]] messages sent by Rebellion players have a chance to be replaced with Combine Propaganda
# Suppression Field

- rate limits API calls for the Rebellion
	- extra slowdown for [[Anti Citizen]]

- costs Points / Minute while active
