# Overwatch keeps track of how rebellious Earth is and reacts accordingly



## 4 Tiered Escalation System

### Watching

- Default State when Overwatch is happy with the state of the occupation

### Suspicious

- [[Anti Citizen]] Behaviour is detected
- [[$REBELLION]] stock is rising
- Health Point Threshold is met
### Alert

- HP Threshold is met

### Hunt Down The Freeman

- HP Threshold is met
- [[Gordon Freeman]] was released from Stasis
